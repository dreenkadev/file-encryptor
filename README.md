# File Encryptor

File encryption utility.

Utilitas enkripsi file.


## Description

Provides secure file encryption and decryption using password-derived keys.

## Deskripsi

Menyediakan enkripsi dan dekripsi file aman menggunakan kunci turunan password.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Encrypt: python file_encrypt.py encrypt file.txt -p password
- Decrypt: python file_encrypt.py decrypt file.txt.enc -p password
- Demo: python file_encrypt.py --demo

## Penggunaan

- Enkripsi: python file_encrypt.py encrypt file.txt -p password
- Dekripsi: python file_encrypt.py decrypt file.txt.enc -p password
- Demo: python file_encrypt.py --demo
